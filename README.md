# Credit Scoring Model

## Objective

This project predicts whether a customer is creditworthy using Machine Learning.

## Dataset

German Credit Dataset

1000 customer records

20 features

## Algorithm

Random Forest Classifier

## Evaluation

Accuracy: 73.5%

ROC-AUC: 0.754

## Libraries

- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Results

The model successfully classified customers into good and bad credit categories and achieved satisfactory performance.
